Introduction

This Quarto document contains the data engineering required to reconstruct and improve the plot made by World Resources Institute, shown in Figure 1, showing the top 10 Green House Gases (GHG) emitters in the world.

The code below requires the following packages:

library(tidyverse)
library(readxl)
library(knitr)
library(gt)
library(dplyr)

# Graphical Visualisation
library(ggplot2)
library(maps)

Conclusion from exploration

Source 1 and Source 2 have similar data, both consisting of each country’s annual GHG emission. These data are suitable for a simple analysis while Source 3 is more detailed, with the data split by industry and gas type.
